Installation Instructions

This guide will help you set up the interactive art installation for OSSO.

Step-by-step Instructions:

1. Server Authorization (Both Computers)

  1. Open Google Chrome on both computers.
  2. Navigate to: https://206.189.10.46:8080
  3. You will be prompted about an insecure connection. Accept and proceed.
  4. This step ensures that Chrome authorizes communication with the server through WSS.

2. Computer 1 - Main Installation (Mini Mac M2)

  1. Attach the camera and ensure it's working properly.
  2. Attach and test the microphone (ensure it's properly charged, keep the second one from the kit charging).
  3. In Chrome, navigate to: https://steam228.github.io/labirintosP5js/labirintosFinal
  4. Once the page is loaded, click in the middle to enter full-screen mode.

3. Computer 2 - Secondary Installation (Mini Mac)

  1. Ensure the projector is properly connected and functioning.
  2. In Chrome, navigate to: https://steam228.github.io/labirintosP5js/labirintosWall
  3. Once the page is loaded, click in the middle to enter full-screen mode.
  4. Position the projector to display the sentences on the designated wall for the poem.
